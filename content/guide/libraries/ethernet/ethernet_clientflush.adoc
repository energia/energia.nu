*EthernetClient*.flush()
------------------------

Discard any bytes that have been written to the client but not yet read.
flush() inherits from the
[Stream](http://energia.nu/reference/stream/ "Stream"){.wikilink}
utility class.

#### Syntax

*client*.flush()

#### Parameters

none
