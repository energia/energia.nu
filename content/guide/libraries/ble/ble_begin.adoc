== ble.begin() ==


=== Description ===

Initializes the communication layer with the network processor. The only
method that can be called before this is
link:../ble_setloglevel/[`ble.setLogLevel()`].

=== Syntax ===

ble.begin();

=== Parameters ===

Nothing

=== Returns ===

link:../ble_error/[Status code]
