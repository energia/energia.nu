== WiFi.scanNetworks() ==

=== Description ===

Scans for available WiFi networks and
returns the discovered number

=== Syntax ===

WiFi.scanNetworks();

=== Parameters ===

none

=== Returns ===

byte : number of discovered networks

=== Example ===
