== WiFiUDP.begin() ==

=== Description ===

Initializes the WiFi UDP library and
network settings. Starts WiFiUDP socket,
listening at local port PORT

=== Syntax ===

WiFiUDP.begin(port);

=== Parameters ===

**port**: the local port to listen on (int)

=== Returns ===

1: if successful

0: if there are no sockets available to use
